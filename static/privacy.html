Politique de confidentialité

1. Données collectées

Lors de la création d'un compte, nous collectons : nom d'utilisateur, adresse e-mail et mot de passe (chiffré). Lors des parties, nous enregistrons vos résultats (placement, jetons, rounds) et faisons évoluer votre classement ELO. Les joueurs invités ne font l'objet d'aucune collecte nominative.

2. Finalité

Vos données sont utilisées exclusivement pour le fonctionnement du jeu : authentification, classements, historique de parties. Elles ne sont ni vendues, ni partagées avec des tiers.

3. Conservation

Les données de compte sont conservées tant que votre compte est actif. Les comptes inactifs depuis plus de 2 ans font l'objet d'un contrôle de rétention. L'historique de parties est conservé même après suppression de compte, mais dissocié de toute identité (pseudonymisation).

4. Vos droits (RGPD)

  • Accès & portabilité (Art. 15 & 20) : Télécharger mes données
  • Rectification (Art. 16) : modifiez vos informations dans votre profil
  • Suppression (Art. 17) : demande de suppression depuis votre profil — délai de grâce de 30 jours
  • Opposition (Art. 21) : contactez-nous via GitHub Issues

5. Sécurité

Les mots de passe sont chiffrés avec bcrypt. Les communications sont chiffrées via HTTPS/WSS en production. Les jetons d'authentification (JWT) expirent après 30 minutes.

6. Contact

Pour toute question relative à vos données personnelles : ouvrez une issue sur le dépôt GitHub du projet.
